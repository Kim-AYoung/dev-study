# dev-study
개발 학습 내용과 질문을 기록하는 곳

## Rule
- 학습 내용은 각 과목명에 해당하는 폴더에 자유롭게 업로드 합시다.
  - 작성자 분류를 위해 파일 앞에 prefix로 본인의 이름을 작성해요.
  - 내용 분류를 위해 하위 폴더를 생성해도 괜찮아요.
- 질문은 이슈 탭에 정리합시다.
  - Assignee으로 할당된 사람은 질문에 대한 답변을 댓글로 남깁니다.
  - 질문자는 질문이 해결되면 이슈룰 closed 해요.
